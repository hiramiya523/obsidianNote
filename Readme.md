# 📖 このVaultについて

技術メモ・ナレッジベースを管理するObsidian Vaultです。

## 使い方

1. **[[Index|🏠 Index]]** から始めてください
2. 各ドキュメント間は内部リンクで移動できます
3. タグやグラフビューで関連情報を探索できます

## Obsidianの機能

このVaultでは以下のObsidian機能を活用しています：

| 機能 | 説明 | 例 |
|------|------|-----|
| **内部リンク** | ドキュメント間のナビゲーション | `[[ページ名]]` |
| **フロントマター** | メタデータ管理 | title, tags, aliases |
| **Callout** | 重要情報の視覚的強調 | `> [!warning]` |
| **タグ** | カテゴリ分類 | #docker #linux |

## カテゴリ

| フォルダ | 内容 |
|---------|------|
| `Docker/` | Docker, Fluent Bit, ログ管理 |
| `Linux/` | Linuxコマンド, サーバー設定 |
| `Database/` | SQL, パフォーマンス |
| `Projects/` | プロジェクト固有の計画 |

## Calloutの凡例

> [!info] 情報
> 補足説明

> [!tip] ヒント
> 推奨事項

> [!warning] 注意
> 注意すべき点

> [!danger] 危険
> 重大な注意点

> [!bug] バグ・問題
> 既知の問題
